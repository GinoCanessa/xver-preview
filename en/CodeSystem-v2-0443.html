CodeSystem: ProviderRole

2.3.0
Official URL: http://terminology.hl7.org/CodeSystem/v2-0443 Version:
Standards status: Trial-use Active as of 2019-11-30 Maturity Level: 0 Responsible: Patient Administration Computable Name: ProviderRole

Copyright/Legal: Copyright HL7. Licensed under creative commons public domain

Code system of concepts specifying the functional involvement with the activity being transmitted (e.g., Case Manager, Evaluator, Transcriber, Nurse Care Practitioner, Midwife, Physician Assistant, etc.). Used in HL7 Version 2.x messaging in the ROL segment.
