Resource Profile: ProfilePaymentNotice

Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-PaymentNotice Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-17 Maturity Level: 0 Computable Name: ProfilePaymentNotice

This cross-version profile allows R5 PaymentNotice content to be represented via FHIR R4 PaymentNotice resources.

Usages:
