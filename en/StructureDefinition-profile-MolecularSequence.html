Resource Profile: ProfileMolecularSequence

Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-MolecularSequence Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-17 Maturity Level: 0 Computable Name: ProfileMolecularSequence

This cross-version profile allows R5 MolecularSequence content to be represented via FHIR R4 MolecularSequence resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.MolecularSequence.relative` for use in FHIR R4, Cross-version Extension `R5.Observation.component` for use in FHIR R4 and Cross-version Extension `R5.Observation.value[x]` for use in FHIR R4

You can also check for usages in the FHIR IG Statistics
