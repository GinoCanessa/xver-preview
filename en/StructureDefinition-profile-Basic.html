Resource Profile: ProfileBasic

Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-Basic Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-17 Maturity Level: 0 Computable Name: ProfileBasic

This cross-version profile allows R5 Basic content to be represented via FHIR R4 Basic resources.

Usages:

  • This Profile is not used by any profiles in this Implementation Guide
