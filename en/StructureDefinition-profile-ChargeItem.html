Resource Profile: ProfileChargeItem

Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-ChargeItem Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-17 Maturity Level: 0 Computable Name: ProfileChargeItem

This cross-version profile allows R5 ChargeItem content to be represented via FHIR R4 ChargeItem resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.Invoice.lineItem` for use in FHIR R4 and Cross-version Extension `R5.PaymentReconciliation.allocation` for use in FHIR R4

You can also check for usages in the FHIR IG Statistics
