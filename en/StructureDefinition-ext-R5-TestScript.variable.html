Official URL: http://hl7.org/fhir/5.0/StructureDefinition/extension-TestScript.variable Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-17 Maturity Level: 0 Computable Name: ExtensionTestScript_Variable

This cross-version extension represents the FHIR R5 element TestScript.variable for use in FHIR R4.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: TestScript.variable

    This extension is allowed for use with FHIR versions R4 to R4 .

Summary

Complex Extension: This cross-version extension represents the FHIR R5 element TestScript.variable for use in FHIR R4.

  • name: string: Descriptive name for this variable.
  • defaultValue: string: A default, hard-coded, or user-defined value for this variable.
  • description: string: A free text natural language description of the variable and its purpose.
  • expression: string: The FHIRPath expression for a specific value to evaluate against the fixture body. When variables are defined, only one of either expression, headerField or path must be specified.
  • headerField: string: Will be used to grab the HTTP header field value from the headers that sourceId is pointing to.
  • hint: string: Displayable text string with hint help information to the user when entering a default value.
  • path: string: XPath or JSONPath to evaluate against the fixture body. When variables are defined, only one of either expression, headerField or path must be specified.
  • sourceId: id: Fixture to evaluate the XPath/JSONPath expression or the headerField against within this variable.
