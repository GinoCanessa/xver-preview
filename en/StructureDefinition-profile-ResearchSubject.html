Resource Profile: ProfileResearchSubject

Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-ResearchSubject Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-17 Maturity Level: 0 Computable Name: ProfileResearchSubject

This cross-version profile allows R5 ResearchSubject content to be represented via FHIR R4 ResearchSubject resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.AdverseEvent.participant` for use in FHIR R4 and Cross-version Extension `R5.AdverseEvent.suspectEntity` for use in FHIR R4
