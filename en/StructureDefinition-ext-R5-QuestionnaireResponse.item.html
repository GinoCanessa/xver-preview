Extension: Cross-version Extension `R5.QuestionnaireResponse.item` for use in FHIR R4

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/extension-QuestionnaireResponse.item Version:
Standards status: Trial-use Active as of 2026-02-06 Maturity Level: 0 Computable Name: ExtensionQuestionnaireResponse_Item

This cross-version extension represents the FHIR R5 element QuestionnaireResponse.item for use in FHIR R4.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: Extension.extension

    This extension is allowed for use with FHIR versions R4 to R4 .

Usage info

Usages:

  • This Extension is not used by any profiles in this Implementation Guide

Summary

Complex Extension: This cross-version extension represents the FHIR R5 element QuestionnaireResponse.item for use in FHIR R4.

  • linkId: string: The item from the Questionnaire that corresponds to this item in the QuestionnaireResponse resource.
  • definition: uri: A reference to an ElementDefinition that provides the details for the item.
  • text: string: Text that is displayed above the contents of the group or as the text of the question being answered.
  • value: boolean, decimal, integer, date, dateTime, time, string, uri, Attachment, Coding, Quantity, Reference: The answer (or one of the answers) provided by the respondent to the question.
  • item: base64Binary, boolean, canonical, code, date, dateTime, decimal, id, instant, integer, markdown, oid, positiveInt, string, time, unsignedInt, uri, url, uuid, Address, Age, Annotation, Attachment, CodeableConcept, Coding, ContactPoint, Count, Distance, Duration, HumanName, Identifier, Money, Period, Quantity, Range, Ratio, Reference, SampledData, Signature, Timing, ContactDetail, Contributor, DataRequirement, Expression, ParameterDefinition, RelatedArtifact, TriggerDefinition, UsageContext, Dosage, Meta: Nested groups and/or questions found within this particular answer.
  • item: base64Binary, boolean, canonical, code, date, dateTime, decimal, id, instant, integer, markdown, oid, positiveInt, string, time, unsignedInt, uri, url, uuid, Address, Age, Annotation, Attachment, CodeableConcept, Coding, ContactPoint, Count, Distance, Duration, HumanName, Identifier, Money, Period, Quantity, Range, Ratio, Reference, SampledData, Signature, Timing, ContactDetail, Contributor, DataRequirement, Expression, ParameterDefinition, RelatedArtifact, TriggerDefinition, UsageContext, Dosage, Meta: Sub-questions, sub-groups or display items nested beneath a group.
