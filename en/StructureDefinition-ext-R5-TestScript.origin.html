Extension: Cross-version Extension `R5.TestScript.origin` for use in FHIR R4

Official URL: http://hl7.org/fhir/5.0/StructureDefinition/extension-TestScript.origin Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-17 Maturity Level: 0 Computable Name: ExtensionTestScript_Origin

This cross-version extension represents the FHIR R5 element TestScript.origin for use in FHIR R4.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: TestScript.origin

    This extension is allowed for use with FHIR versions R4 to R4 .

Summary

Complex Extension: This cross-version extension represents the FHIR R5 element TestScript.origin for use in FHIR R4.

  • index: integer: Abstract name given to an origin server in this test script. The name is provided as a number starting at 1.
  • profile: Coding: The type of origin profile the test system supports.
  • url: url: The explicit url path of the origin server used in this test script.
