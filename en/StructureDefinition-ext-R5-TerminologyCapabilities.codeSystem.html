Summary

Complex Extension: This cross-version extension represents the FHIR R5 element TerminologyCapabilities.codeSystem for use in FHIR R4.

  • uri: canonical: Canonical identifier for the code system, represented as a URI.
  • code: string: For version-less code systems, there should be a single version with no identifier.
  • isDefault: boolean: If this is the default version for this code system.
  • compositional: boolean: If the compositional grammar defined by the code system is supported.
  • language: code: Language Displays supported.
  • code: code: Code of the property supported.
  • op: code: Operations supported for the property.
  • property: code: Properties supported for $lookup.
  • content: code: The extent of the content of the code system (the concepts and codes it defines) are represented in this resource instance.
  • subsumption: boolean: True if subsumption is supported for this version of the code system.

Maturity: 0

Summary

Complex Extension: This cross-version extension represents the FHIR R5 element TerminologyCapabilities.codeSystem for use in FHIR R4.

  • uri: canonical: Canonical identifier for the code system, represented as a URI.
  • code: string: For version-less code systems, there should be a single version with no identifier.
  • isDefault: boolean: If this is the default version for this code system.
  • compositional: boolean: If the compositional grammar defined by the code system is supported.
  • language: code: Language Displays supported.
  • code: code: Code of the property supported.
  • op: code: Operations supported for the property.
  • property: code: Properties supported for $lookup.
  • content: code: The extent of the content of the code system (the concepts and codes it defines) are represented in this resource instance.
  • subsumption: boolean: True if subsumption is supported for this version of the code system.
