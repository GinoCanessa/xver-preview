CodeSystem: ColorCodes

5.0.0
Official URL: http://hl7.org/fhir/color-names Version:
Standards status: Trial-use Draft as of 2026-02-06 Maturity Level: 0 Computable Name: ColorCodes

This work is distributed under the W3C® Software and Document License in the hope that it will be useful, but WITHOUT ANY WARRANTY; without even the implied warranty of MERCHANTABILITY or FITNESS FOR A PARTICULAR PURPOSE.

This code system represents that named RGB colors found in the CSS4 specification. The names are not case sensitive and different cases are encountered in common use (e.g. AliceBlue vs aliceblue.

This Code system is referenced in the definition of the following value sets:

  • This CodeSystem is not used here; it may be used elsewhere (e.g. specifications and/or implementations that use this content)

Properties

This code system defines the following properties for its concepts

NameCodeTypeDescription
rgb rgb string The RGB color that matches the named color (as defined in CSS4 specification)
