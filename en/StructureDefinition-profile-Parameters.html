Resource Profile: ProfileParameters

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-Parameters Version:
Standards status: Trial-use Active as of 2026-02-09 Maturity Level: 0 Computable Name: ProfileParameters

This cross-version profile allows R5 Parameters content to be represented via FHIR R4 Parameters resources.

Usages:

  • This Profile is not used by any profiles in this Implementation Guide
