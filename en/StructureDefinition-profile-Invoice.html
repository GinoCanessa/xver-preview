Resource Profile: ProfileInvoice

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-Invoice Version:
Standards status: Trial-use Active as of 2026-02-06 Maturity Level: 0 Computable Name: ProfileInvoice

This cross-version profile allows R5 Invoice content to be represented via FHIR R4 Invoice resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.PaymentReconciliation.allocation.target` for use in FHIR R4
