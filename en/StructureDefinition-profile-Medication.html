Resource Profile: ProfileMedication

Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-Medication Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-17 Maturity Level: 0 Computable Name: ProfileMedication

This cross-version profile allows R5 Medication content to be represented via FHIR R4 Medication resources.
