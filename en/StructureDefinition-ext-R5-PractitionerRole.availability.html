Standards status: Trial-use Active as of 2026-02-06 Maturity Level: 0 Computable Name: ExtensionPractitionerRole_Availability

This cross-version extension represents the FHIR R5 element PractitionerRole.availability for use in FHIR R4.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: Element

    This extension is allowed for use with FHIR versions R4 to R4 .

Summary

Complex Extension: This cross-version extension represents the FHIR R5 element PractitionerRole.availability for use in FHIR R4.

  • _datatype: string: Slice to indicate the presence of a R5 Availability in FHIR R4
  • daysOfWeek: code: mon | tue | wed | thu | fri | sat | sun.
  • allDay: boolean: Always available? i.e. 24 hour service.
  • availableStartTime: time: Opening time of day (ignored if allDay = true).
  • availableEndTime: time: Closing time of day (ignored if allDay = true).
  • description: string: Reason presented to the user explaining why time not available.
  • during: Period: Service not available during this period.

Maturity: 0
