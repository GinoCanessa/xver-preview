Resource Profile: ProfileGraphDefinition

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-GraphDefinition Version:
Standards status: Trial-use Active as of 2026-02-06 Maturity Level: 0 Computable Name: ProfileGraphDefinition

This cross-version profile allows R5 GraphDefinition content to be represented via FHIR R4 GraphDefinition resources.

Usages:

  • This Profile is not used by any profiles in this Implementation Guide

You can also check for usages in the FHIR IG Statistics
