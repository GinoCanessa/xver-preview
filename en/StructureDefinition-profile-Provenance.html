Resource Profile: ProfileProvenance

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-Provenance Version:
Standards status: Trial-use Active as of 2026-02-06 Maturity Level: 0 Computable Name: ProfileProvenance

This cross-version profile allows R5 Provenance content to be represented via FHIR R4 Provenance resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.DeviceDispense` for use in FHIR R4 and Cross-version Extension `R5.Transport` for use in FHIR R4

You can also check for usages in the FHIR IG Statistics
