Resource Profile: ProfileDiagnosticReport

Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-DiagnosticReport Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-21 Maturity Level: 0 Computable Name: ProfileDiagnosticReport

This cross-version profile allows R5 DiagnosticReport content to be represented via FHIR R4 DiagnosticReport resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.ChargeItem.service` for use in FHIR R4, Cross-version Extension `R5.DiagnosticReport.supportingInfo` for use in FHIR R4 and Cross-version Extension `R5.NutritionIntake` for use in FHIR R4

You can also check for usages in the FHIR IG Statistics
