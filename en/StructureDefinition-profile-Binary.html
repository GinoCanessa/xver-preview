Resource Profile: ProfileBinary

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-Binary Version:
Standards status: Trial-use Active as of 2026-02-09 Maturity Level: 0 Computable Name: ProfileBinary

This cross-version profile allows R5 Binary content to be represented via FHIR R4 Binary resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.AdministrableProductDefinition` for use in FHIR R4 and Cross-version Extension `R5.Appointment.patientInstruction` for use in FHIR R4

You can also check for usages in the FHIR IG Statistics
