Resource Profile: ProfileEndpoint

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-Endpoint Version:
Standards status: Trial-use Active as of 2026-02-06 Maturity Level: 0 Computable Name: ProfileEndpoint

This cross-version profile allows R5 Endpoint content to be represented via FHIR R4 Endpoint resources.
