Extension: Cross-version Extension `R5.Account.relatedAccount` for use in FHIR R4

Official URL: http://hl7.org/fhir/5.0/StructureDefinition/extension-Account.relatedAccount Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-17 Maturity Level: 0 Computable Name: ExtensionAccount_RelatedAccount

This cross-version extension represents the FHIR R5 element Account.relatedAccount for use in FHIR R4.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: Account

    This extension is allowed for use with FHIR versions R4 to R4 .
