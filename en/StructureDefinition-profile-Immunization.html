Resource Profile: ProfileImmunization

Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-Immunization Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-21 Maturity Level: 0 Computable Name: ProfileImmunization

This cross-version profile allows R5 Immunization content to be represented via FHIR R4 Immunization resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.AdverseEvent.contributingFactor` for use in FHIR R4, Cross-version Extension `R5.AdverseEvent.preventiveAction` for use in FHIR R4, Cross-version Extension `R5.AdverseEvent.supportingInfo` for use in FHIR R4 and Cross-version Extension `R5.ChargeItem.service` for use in FHIR R4

You can also check for usages in the FHIR IG Statistics
