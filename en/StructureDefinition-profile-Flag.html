Resource Profile: ProfileFlag

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-Flag Version:
Standards status: Trial-use Active as of 2026-02-09 Maturity Level: 0 Computable Name: ProfileFlag

This cross-version profile allows R5 Flag content to be represented via FHIR R4 Flag resources.

Usages:

  • This Profile is not used by any profiles in this Implementation Guide
