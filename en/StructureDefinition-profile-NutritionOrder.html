Resource Profile: ProfileNutritionOrder

Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-NutritionOrder Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-17 Maturity Level: 0 Computable Name: ProfileNutritionOrder

This cross-version profile allows R5 NutritionOrder content to be represented via FHIR R4 NutritionOrder resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.Claim.item` for use in FHIR R4, Cross-version Extension `R5.ClaimResponse.addItem` for use in FHIR R4, Cross-version Extension `R5.ExplanationOfBenefit.addItem` for use in FHIR R4, Cross-version Extension `R5.ExplanationOfBenefit.item` for use in FHIR R4 and Cross-version Extension `R5.NutritionIntake` for use in FHIR R4

You can also check for usages in the FHIR IG Statistics
