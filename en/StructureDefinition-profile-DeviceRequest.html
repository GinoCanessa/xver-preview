Resource Profile: ProfileDeviceRequest

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-DeviceRequest Version:
Standards status: Trial-use Active as of 2026-02-06 Maturity Level: 0 Computable Name: ProfileDeviceRequest

This cross-version profile allows R5 DeviceRequest content to be represented via FHIR R4 DeviceRequest resources.
