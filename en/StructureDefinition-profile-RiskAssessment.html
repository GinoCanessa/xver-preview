Resource Profile: ProfileRiskAssessment

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-RiskAssessment Version:
Standards status: Trial-use Active as of 2026-02-06 Maturity Level: 0 Computable Name: ProfileRiskAssessment

This cross-version profile allows R5 RiskAssessment content to be represented via FHIR R4 RiskAssessment resources.

Usages:
