Resource Profile: ProfileAdministrableProductDefinition

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-AdministrableProductDefinition Version:
Standards status: Trial-use Active as of 2026-02-09 Maturity Level: 0 Computable Name: ProfileAdministrableProductDefinition

This cross-version profile allows R5 AdministrableProductDefinition content to be represented via FHIR R4 Basic resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.ActivityDefinition.subject[x]` for use in FHIR R4, Cross-version Extension `R5.Ingredient` for use in FHIR R4 and Cross-version Extension `R5.PlanDefinition.subject[x]` for use in FHIR R4

You can also check for usages in the FHIR IG Statistics
