Resource Profile: ProfileQuestionnaire

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-Questionnaire Version:
Standards status: Trial-use Active as of 2026-02-06 Maturity Level: 0 Computable Name: ProfileQuestionnaire

This cross-version profile allows R5 Questionnaire content to be represented via FHIR R4 Questionnaire resources.
