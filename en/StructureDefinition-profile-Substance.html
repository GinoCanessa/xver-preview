Resource Profile: ProfileSubstance

Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-Substance Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-21 Maturity Level: 0 Computable Name: ProfileSubstance

This cross-version profile allows R5 Substance content to be represented via FHIR R4 Substance resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.ClinicalUseDefinition` for use in FHIR R4, Cross-version Extension `R5.NutritionIntake` for use in FHIR R4, Cross-version Extension `R5.NutritionProduct` for use in FHIR R4, Cross-version Extension `R5.ServiceRequest.orderDetail` for use in FHIR R4 and Cross-version Extension `R5.ClinicalUseDefinition.contraindication.otherTherapy` for use in FHIR R4

You can also check for usages in the FHIR IG Statistics
