Standards status: Trial-use Active as of 2026-02-06 Maturity Level: 0 Computable Name: ExtensionExtendedContactDetail

This cross-version extension represents the FHIR R5 element ExtendedContactDetail for use in FHIR R4.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: Basic

    This extension is allowed for use with FHIR versions R4 to R4 .

Summary

Complex Extension: This cross-version extension represents the FHIR R5 element ExtendedContactDetail for use in FHIR R4.

  • purpose: CodeableConcept: The purpose/type of contact.
  • name: HumanName: The name of an individual to contact, some types of contact detail are usually blank.
  • telecom: ContactPoint: The contact details application for the purpose defined.
  • address: Address: Address for the contact.
  • organization: Reference: This contact detail is handled/monitored by a specific organization. If the name is provided in the contact, then it is referring to the named individual within this organization.
  • period: Period: Period that this contact was valid for usage.
