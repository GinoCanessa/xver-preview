Summary

Complex Extension: This cross-version extension represents the FHIR R5 element MedicationKnowledge.storageGuideline for use in FHIR R4.

  • reference: uri: Reference to additional information about the storage guidelines.
  • note: Annotation: Additional notes about the storage.
  • stabilityDuration: Duration: Duration that the medication remains stable if the environmentalSetting is respected.
  • type: CodeableConcept: Identifies the category or type of setting (e.g., type of location, temperature, humidity).
  • value: Quantity, Range, CodeableConcept: Value associated to the setting. E.g., 40° – 50°F for temperature.

Maturity: 0
