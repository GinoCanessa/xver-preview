Resource Profile: ProfileDeviceMetric

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-DeviceMetric Version:
Standards status: Trial-use Active as of 2026-02-06 Maturity Level: 0 Computable Name: ProfileDeviceMetric

This cross-version profile allows R5 DeviceMetric content to be represented via FHIR R4 DeviceMetric resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.EvidenceVariable.characteristic.definitionByTypeAndValue` for use in FHIR R4 and Cross-version Extension `R5.EvidenceVariable.characteristic` for use in FHIR R4

You can also check for usages in the FHIR IG Statistics
