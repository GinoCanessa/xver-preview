Resource Profile: ProfileMessageDefinition

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-MessageDefinition Version:
Standards status: Trial-use Active as of 2026-02-09 Maturity Level: 0 Computable Name: ProfileMessageDefinition

This cross-version profile allows R5 MessageDefinition content to be represented via FHIR R4 MessageDefinition resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.NutritionIntake` for use in FHIR R4 and Cross-version Extension `R5.PlanDefinition.action` for use in FHIR R4
