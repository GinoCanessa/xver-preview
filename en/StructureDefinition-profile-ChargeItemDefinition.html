Resource Profile: ProfileChargeItemDefinition

Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-ChargeItemDefinition Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-21 Maturity Level: 0 Computable Name: ProfileChargeItemDefinition

This cross-version profile allows R5 ChargeItemDefinition content to be represented via FHIR R4 ChargeItemDefinition resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.DeviceDefinition.chargeItem` for use in FHIR R4 and Cross-version Extension `R5.NutritionIntake` for use in FHIR R4
