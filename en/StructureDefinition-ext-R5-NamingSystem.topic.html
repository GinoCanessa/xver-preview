Extension: Cross-version Extension `R5.NamingSystem.topic` for use in FHIR R4

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/extension-NamingSystem.topic Version:
Standards status: Trial-use Active as of 2026-02-10 Maturity Level: 0 Computable Name: ExtensionNamingSystem_Topic

This cross-version extension represents the FHIR R5 element NamingSystem.topic for use in FHIR R4.

Context of Use

    This extension may be used on the following element(s)

    • Element ID: NamingSystem

    This extension is allowed for use with FHIR versions R4 to R4 .
