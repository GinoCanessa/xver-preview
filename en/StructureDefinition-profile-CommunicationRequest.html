Resource Profile: ProfileCommunicationRequest

0.0.1-snapshot-3
Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-CommunicationRequest Version:
Standards status: Trial-use Active as of 2026-02-06 Maturity Level: 0 Computable Name: ProfileCommunicationRequest

This cross-version profile allows R5 CommunicationRequest content to be represented via FHIR R4 CommunicationRequest resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.DocumentReference.basedOn` for use in FHIR R4 and Cross-version Extension `R5.GuidanceResponse.result` for use in FHIR R4

You can also check for usages in the FHIR IG Statistics
