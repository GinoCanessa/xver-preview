Resource Profile: ProfileServiceRequest

Official URL: http://hl7.org/fhir/5.0/StructureDefinition/profile-ServiceRequest Version: 0.0.1-snapshot-3
Standards status: Trial-use Active as of 2026-02-21 Maturity Level: 0 Computable Name: ProfileServiceRequest

This cross-version profile allows R5 ServiceRequest content to be represented via FHIR R4 ServiceRequest resources.

Usages:

  • Refer to this Profile: Cross-version Extension `R5.BiologicallyDerivedProductDispense` for use in FHIR R4, Cross-version Extension `R5.ChargeItem.service` for use in FHIR R4, Cross-version Extension `R5.GenomicStudy` for use in FHIR R4, Cross-version Extension `R5.ImagingSelection` for use in FHIR R4 and Cross-version Extension `R5.NutritionIntake` for use in FHIR R4

You can also check for usages in the FHIR IG Statistics
